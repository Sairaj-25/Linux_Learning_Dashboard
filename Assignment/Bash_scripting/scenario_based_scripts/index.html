🖥️ 50 BASH SCRIPTING PROBLEMS

Beginner to Advanced Solutions & Explanations

Beginner (1-10) Beg-Int (11-20) Intermediate (21-30) Int-Adv (31-40) Advanced (41-50) Interview Q&A

BEGINNER LEVEL (1-10)

1. Check File Existence
#!/bin/bash
read -p "Enter filename: " file
if [ -f "$file" ]; then echo "Regular file."
elif [ -d "$file" ]; then echo "Directory."
else echo "Missing." fi

Flow: Use test operators -f (file) and -d (directory) to determine type. Alt: test -e or [[ -f $file ]].

2. Count Lines in a File
#!/bin/bash
read -p "Enter filename: " file
[ ! -r "$file" ] && echo "File missing/unreadable" && exit 1
wc -l < "$file"

Flow: Validate readability with -r, then wc -l. Alt: cat -n "$file" | tail -1 | awk '{print $1}'.

3. Count Words in a File
#!/bin/bash
read -p "Enter filename: " file
[ ! -r "$file" ] && echo "File missing/unreadable" && exit 1
wc -w < "$file"

Flow: wc -w counts words. Alt: tr ' ' '\n' < "$file" | wc -l.

4. Rename Files by Extension
#!/bin/bash
for f in *.txt; do
  [ -f "$f" ] && mv "$f" "$f.bak"
done

Flow: Glob *.txt and loop. Alt: rename '.txt' '.txt.bak' *.txt.

5. Display First N Lines
#!/bin/bash
[ $# -ne 2 ] && echo "Usage: $0 file N" && exit 1
head -n "$2" "$1"

Flow: Validate args, use head. Alt: sed -n "1,$2p" "$1".

6. Display Last N Lines
#!/bin/bash
[ $# -ne 2 ] && echo "Usage: $0 file N" && exit 1
tail -n "$2" "$1"

Flow: Use tail. Alt: sed -n "$(($(wc -l < $1)-$2+1),\$p" "$1".

7. Check Even or Odd Number
#!/bin/bash
read -p "Enter number: " num
[[ ! $num =~ ^[0-9]+$ ]] && echo "Invalid" && exit 1
[ $((num % 2)) -eq 0 ] && echo "Even" || echo "Odd"

Flow: Regex validation, modulo operator %. Alt: if (( num % 2 )).

8. Find the Largest of Three Numbers
#!/bin/bash
read -p "Enter 3 nums: " a b c
largest=$a
[ $b -gt $largest ] && largest=$b
[ $c -gt $largest ] && largest=$c
echo "Largest: $largest"

Flow: Sequential comparison. Alt: echo -e "$a\n$b\n$c" | sort -n | tail -1.

9. Print Multiplication Table
#!/bin/bash
read -p "Enter number: " n
[[ ! $n =~ ^[1-9][0-9]*$ ]] && echo "Invalid" && exit 1
for i in {1..10}; do echo "$n x $i = $((n*i))"; done

Flow: Loop 1-10, validate positive integer. Alt: seq 1 10 | xargs -I{} echo "$n x {} = $((n*{}))".

10. Check Prime Number
#!/bin/bash
read -p "Enter number: " n
[ $n -le 1 ] && echo "Not Prime" && exit 0
for ((i=2; i<=n/2; i++)); do [ $((n%i)) -eq 0 ] && echo "Not Prime" && exit 0; done
echo "Prime"

Flow: Check divisibility up to n/2. Alt: factor $n | awk '{print NF==2 ? "Prime" : "Not Prime"}'.

BEGINNER TO INTERMEDIATE (11-20)

11. Create Multiple Directories
#!/bin/bash
read -p "Prefix & Count: " prefix count
for ((i=1; i<=count; i++)); do
  [ -d "${prefix}_${i}" ] && continue
  mkdir "${prefix}_${i}"
done

Flow: Loop and check existence. Alt: mkdir ${prefix}_{1..$count} (Brace expansion eval required).

12. Backup a File
#!/bin/bash
[ -z "$1" ] && echo "Need file" && exit 1
cp "$1" "${1}_$(date +%Y%m%d_%H%M%S).bak"

Flow: Timestamp via date. Alt: cp $1 backup/${1##*/}.bak.

13. Delete Empty Files
#!/bin/bash
dir=${1:-.}
find "$dir" -maxdepth 1 -type f -empty -print -delete

Flow: find -empty -delete. Alt: for f in *; do [ ! -s "$f" ] && rm "$f"; done.

14. Find Files Larger Than Given Size
#!/bin/bash
[ $# -ne 2 ] && echo "Usage: $0 Dir MB" && exit 1
[ ! -d "$1" ] && echo "Dir missing" && exit 1
find "$1" -type f -size +"${2}M"

Flow: find -size. Alt: du -ah "$1" | awk '$1 > $2"MB"'.

15. Count Files by Extension
#!/bin/bash
[ $# -ne 2 ] && echo "Usage: $0 Dir Ext" && exit 1
find "$1" -maxdepth 1 -type f -name "*.$2" | wc -l

Flow: find -name limit depth 1. Alt: ls -1 *.$2 2>/dev/null | wc -l.

16. Find Duplicate Lines in a File
#!/bin/bash
[ ! -f "$1" ] && exit 1
grep -v '^$' "$1" | sort | uniq -d

Flow: sort then uniq -d. Alt: awk '{count[$0]++} count[$0]==2{print}' "$1".

17. Remove Blank Lines from a File
#!/bin/bash
grep -v '^$' "$1" > "${1}.cleaned"

Flow: grep -v inverse match blank. Alt: sed '/^$/d' "$1" > out.

18. Search Word in File
#!/bin/bash
[ $# -ne 2 ] && exit 1
grep -c "$1" "$2"

Flow: grep -c counts. Alt: awk '/word/{c++} END{print c}'.

19. Case-Insensitive Word Search
#!/bin/bash
[ $# -ne 2 ] && exit 1
grep -in "$1" "$2"

Flow: -i ignore case, -n line numbers. Alt: awk 'BEGIN{IGNORECASE=1} /word/{print NR": "$0}'.

20. Replace Word in File
#!/bin/bash
[ $# -ne 3 ] && exit 1
sed "s/$1/$2/g" "$3" > "${3}.mod"

Flow: sed s///g. Alt: awk '{gsub(a,b); print}' a="$1" b="$2" "$3" > out.

INTERMEDIATE LEVEL (21-30)

21. Parse CSV and Print Specific Column
#!/bin/bash
[ $# -ne 2 ] && exit 1
awk -F, -v col="$2" '{print $col}' "$1"

Flow: awk -F, sets comma delimiter. Alt: cut -d, -f$2 "$1".

22. Sum Amounts from CSV by Username
#!/bin/bash
[ $# -ne 2 ] && exit 1
awk -F, -v user="$1" 'NR>1 && $1==user{sum+=$2} END{print sum}' "$2"

Flow: Skip header NR>1, match user, sum. Alt: grep "$1" "$2" | tail -n +2 | awk -F, '{sum+=$2} END{print sum}'.

23. Validate Email Format
#!/bin/bash
read -p "Email: " email
regex="^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$"
[[ $email =~ $regex ]] && echo "Valid" || echo "Invalid"

Flow: Bash regex [[ =~ ]]. Alt: grep -qE 'regex' <<< "$email".

24. Validate Password Strength
#!/bin/bash
read -s -p "Password: " pass
[[ ${#pass} -lt 8 ]] && echo "Need 8 chars" && exit 1
[[ ! "$pass" =~ [A-Z] ]] && echo "Need Uppercase" && exit 1
[[ ! "$pass" =~ [a-z] ]] && echo "Need Lowercase" && exit 1
[[ ! "$pass" =~ [0-9] ]] && echo "Need Digit" && exit 1
[[ ! "$pass" =~ [^a-zA-Z0-9] ]] && echo "Need Special" && exit 1
echo "Strong"

Flow: Check length, then regex for char classes. Alt: cracklib-check or pwscore.

25. Monitor Disk Usage
#!/bin/bash
usage=$(df -h / | awk 'NR==2{print $5}' | tr -d '%')
[ "$usage" -gt 80 ] && echo "ALERT: / is ${usage}% full"

Flow: Parse df, strip %, compare. Alt: stat -f /.

26. Check Service Status
#!/bin/bash
[ -z "$1" ] && exit 1
systemctl is-active --quiet "$1" && echo "$1 Active" || echo "$1 Inactive"

Flow: systemctl is-active. Alt: service "$1" status.

27. Restart Failed Service
#!/bin/bash
svc="$1"
! systemctl is-active --quiet "$svc" && systemctl restart "$svc" && echo "$(date): Restarted $svc" >> /var/log/svc_monitor.log

Flow: Check, restart, log with timestamp. Alt: /etc/init.d/$1 restart.

28. Check Process by Name
#!/bin/bash
[ -z "$1" ] && exit 1
pid=$(pgrep -x "$1")
[ -n "$pid" ] && echo "Running PID: $pid" || echo "Not running"

Flow: pgrep -x exact match. Alt: ps -C "$1" -o pid=.

29. Kill Process by Name
#!/bin/bash
pids=$(pgrep "$1")
[ -z "$pids" ] && echo "Not found" && exit 0
echo "Found: $pids. Kill? (y/n)"; read ans
[[ $ans == "y" ]] && kill -9 $pids

Flow: pgrep, confirm, kill -9. Alt: pkill -i "$1".

30. Show Top Memory-Consuming Processes
#!/bin/bash
ps -eo pid,user,%mem,comm --sort=-%mem | head -n 6

Flow: ps --sort, limit 6 (header + 5). Alt: top -b -o +%MEM | head -n 12.

INTERMEDIATE TO ADVANCED (31-40)

31. Log File Analyzer
#!/bin/bash
echo "ERROR: $(grep -c 'ERROR' $1)"
echo "WARN: $(grep -c 'WARNING' $1)"
echo "INFO: $(grep -c 'INFO' $1)"

Flow: grep -c for counts. Alt: awk '/ERROR/{e++} /WARNING/{w++} /INFO/{i++} END{print e,w,i}' $1.

32. Extract IP Addresses from Logs
#!/bin/bash
grep -oE '([0-9]{1,3}\.){3}[0-9]{1,3}' "$1" | sort -u

Flow: grep -oE extracts matching pattern. Alt: awk '{for(i=1;i<=NF;i++) if($i~/^([0-9]+\.){3}[0-9]+/) print $i}' $1 | sort -u.

33. Count Requests Per IP
#!/bin/bash
awk '{print $1}' "$1" | sort | uniq -c | sort -nr

Flow: Extract IP (field 1 usually), sort, uniq count. Alt: cat "$1" | cut -d' ' -f1 | sort | uniq -c | sort -nr.

#!/bin/bash

35. Archive Old Files
#!/bin/bash
dir=$1; archive="$2"
mkdir -p "$archive"
find "$dir" -maxdepth 1 -type f -mtime +30 -exec mv {} "$archive" \;

Flow: find -mtime +30. Alt: for f in $(find ...); do mv $f $archive; done.

36. Compress Log Files
#!/bin/bash
find "$1" -name "*.log" -mtime +7 ! -name "*.gz" -exec gzip {} \;

Flow: Find logs >7 days, exclude .gz, compress. Alt: logrotate is the enterprise solution.

37. Menu-Driven File Utility
#!/bin/bash
while true; do
  read -p "1.Create 2.Rename 3.Delete 4.View 5.Exit: " ch
  case $ch in
    1) read -p "File: " f; touch "$f";;
    2) read -p "Old New: " o n; mv "$o" "$n";;
    3) read -p "File: " f; rm "$f";;
    4) read -p "File: " f; cat "$f";;
    5) exit 0;;
  esac
done

Flow: Infinite loop + case. Alt: select opt in Create Rename Delete View Exit; do ... done.

39. Create Users from File
#!/bin/bash

  id "$user" &>/dev/null && echo "Skipped $user" || { useradd "$user" && echo "Created $user"; }
done < "$1"

Flow: Read file, check existence, create. Alt: newusers command for bulk creation.

40. Check File Permissions
#!/bin/bash

Flow: Test operators. Alt: stat -c "%A" $1 | cut -c2-4.

ADVANCED LEVEL (41-50)

41. Recursive File Statistics
#!/bin/bash
dir=$1
f=$(find "$dir" -type f | wc -l)
d=$(find "$dir" -type d | wc -l)
e=$(find "$dir" -type f -empty | wc -l)
x=$(find "$dir" -type f -executable | wc -l)
echo -e "Files: $f\nDirs: $d\nEmpty: $e\nExec: $x"

Flow: Separate find commands. Alt: Single find + awk counter script.

42. Directory Size Report
#!/bin/bash
du -sh "$1"/*/ 2>/dev/null | sort -hr

Flow: du -sh on subdirs, sort. Alt: find "$1" -maxdepth 1 -type d -exec du -sh {} \; | sort -hr.

43. Safe File Cleanup Script
#!/bin/bash
ext=$1; dir=$2; dry=$3
[ "$dry" = "--dry-run" ] && act="echo rm" || act="rm"
find "$dir" -type f -name "*.$ext" -exec $act {} \;

Flow: Dry-run echoes instead of removing. Alt: find -ok rm {} \; prompts per file.

44. Retry Failed Command
#!/bin/bash
cmd="$@"
for i in {1..5}; do
  $cmd && exit 0
  echo "Retry $i..."
  sleep 2
done
echo "Failed after 5 retries"; exit 1

Flow: Loop 1-5, execute, break on success. Alt: while ! $cmd; do ... done.

45. Lock File to Prevent Duplicate Script Runs
#!/bin/bash
LOCK=/tmp/myscript.lock
[ -f "$LOCK" ] && exit 1
touch "$LOCK"
trap 'rm -f "$LOCK"; exit' EXIT
# Script logic here
sleep 10

Flow: Check lock, create, trap for cleanup. Alt: flock -n "$LOCK" ./script.sh.

46. Trap Script Interruptions
#!/bin/bash
TMP=/tmp/mytemp_$$ touch "$TMP"
trap 'rm -f "$TMP"; echo "Cleaned"; exit' SIGINT SIGTERM
echo "Running... CTRL+C to test"
sleep 100

Flow: trap handles signals. Alt: Trap EXIT signal which catches all exits.

49. Monitor CPU Usage Continuously
#!/bin/bash
while true; do
  idle=$(top -bn1 | grep "Cpu(s)" | awk '{print $8}')
  usage=$(echo "100 - $idle" | bc)
  [ ${usage%.*} -gt 80 ] && echo "ALERT: CPU ${usage}%"
  sleep 5
done

Flow: Parse top, compare. Alt: mpstat 5 1 | awk '/Average/{print 100-$NF}'.
